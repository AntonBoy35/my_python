# assembly_pdf.py

Прпограмма загружает данные из json документа о посещаемости библиотек в различных районах г. Москва, строит круговую диаграмму с информацией о посещаемости в 20ти районах с наиболее популярными библиотеками и создает отчет в pdf формате, состоящий из двух страниц, на первой располагается титульный лист загруженный с сайта, на второй располагается итоговая диаграмма с посещаемостью библиотек, районом с самой большой посещаемостью библиотек и числом посетителей в нем.
